# Binary


## Techniques

**XOR**

XOR of a number and 0 is the number
- a ⊕ 0 = a

XOR of a number and itself is 0
- a ⊕ a = 0

**XOR is associative and communtative**
- a ⊕ b ⊕ a = (a ⊕ a) ⊕ b = 0 ⊕ b = b
